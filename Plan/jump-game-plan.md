## 2. 핵심 기능

### 2.1 주인공 (Jumping Girl)
- 화면 좌측 또는 중앙에 고정된 X 위치
- **점프**: 스페이스 / 클릭 / 터치로 점프
- 중력 + 점프력 적용
- 바닥(플랫폼) 위에서만 재점프 가능
- 간단한 스프라이트 또는 도형으로 표현

### 2.2 세로 스크롤
- 배경/맵이 **아래로** 스크롤 (또는 주인공이 위로 올라가는 느낌)
- 스크롤 속도는 난이도에 따라 조절 가능
- 무한 or 구간별 맵 생성

### 2.3 적 (Food Enemies)
- **종류**: 사과, 햄버거, 피자, 감자튀김 등 (도형/이모지/이미지)
- 화면 위쪽에서 생성 → 아래로 이동 (스크롤 방향과 함께)
- 랜덤 간격, 랜덤 X 위치 (또는 패턴)
- 닿으면 게임 오버

### 2.4 충돌
- 주인공 vs 음식: 충돌 시 **게임 오버**
- (선택) 아이템: 먹으면 점수, 회복 등

## 3. 게임 루프 및 상태

| 상태 | 설명 |
|------|------|
| `start` | 타이틀, "시작" 버튼 |
| `playing` | 플레이 중 (점프, 스크롤, 적 생성) |
| `gameover` | 점수 표시, "다시 하기" |
