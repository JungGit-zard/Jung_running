# Pixel Lab으로 슬라이딩 애니메이션 만들기

**목적**: Jumping Girl 게임에 주인공의 슬라이딩(미끄러지기) 애니메이션을 추가하기 위한 가이드.

---

## 1. Pixel Lab이란?

- **Pixel Lab**은 픽셀 아트와 애니메이션을 만들 수 있는 도구입니다.
- 픽셀 단위로 그림을 그리고, 프레임별로 애니메이션을 만들 수 있습니다.

---

## 2. 슬라이딩 애니메이션 제작 가이드

### 기본 요구사항

- **크기**: `run.mp4`와 비슷한 비율 (약 48×90 픽셀, 또는 그 비율)
- **배경**: **라임 오렌지/녹색** (크로마키용) 또는 **투명 배경**
- **프레임 수**: 4~6프레임 권장 (너무 많으면 파일 크기 증가)
- **형식**: 
  - **비디오**: `slide.mp4` (MP4, H.264)
  - 또는 **이미지 시퀀스**: `slide_01.png`, `slide_02.png`, ... (PNG, 투명 배경)

### 슬라이딩 동작 구성

1. **시작 프레임** (1프레임): 서 있는 자세에서 몸을 낮추기 시작
2. **슬라이딩 중** (2~3프레임): 몸을 낮춰서 미끄러지는 자세
   - 다리가 앞으로 뻗어있고, 몸이 낮아짐
   - 손이 바닥에 닿거나, 몸이 거의 수평에 가까움
3. **끝 프레임** (1프레임): 슬라이딩 종료, 다시 일어서기 시작

### Pixel Lab에서 만드는 방법

#### A) 비디오로 만들기 (권장)

1. **프로젝트 설정**
   - 캔버스 크기: 48×90 (또는 비슷한 비율)
   - 프레임레이트: 10~12fps (슬라이딩은 빠른 동작이므로)
   - 배경색: **라임 오렌지** (`#BFFF00` 또는 `#ADFF2F`) 또는 **녹색** (`#00FF00`)

2. **프레임별 그리기**
   - **프레임 1**: 서 있는 자세 (약간 구부린 상태)
   - **프레임 2**: 몸을 더 낮춤, 다리 시작
   - **프레임 3**: 슬라이딩 자세 (다리 앞으로, 몸 낮음)
   - **프레임 4**: 슬라이딩 유지 (약간 변화)
   - **프레임 5**: 일어서기 시작
   - **프레임 6**: 서 있는 자세로 복귀

3. **내보내기**
   - 형식: **MP4** (H.264)
   - 파일명: `slide.mp4`
   - 저장 경로: `graphic_resource/character/amin/slide.mp4`

#### B) 이미지 시퀀스로 만들기

1. 각 프레임을 **PNG**로 내보내기
2. 파일명: `slide_01.png`, `slide_02.png`, ... `slide_06.png`
3. 저장 경로: `graphic_resource/character/amin/slide_01.png` 등

---

## 3. 게임에 통합하기

### 파일 위치

- **비디오**: `graphic_resource/character/amin/slide.mp4`
- **이미지 시퀀스**: `graphic_resource/character/amin/slide_01.png`, `slide_02.png`, ...

### 입력 키

- **S키** 또는 **아래 화살표(ArrowDown)**: 슬라이딩 시작
- 슬라이딩 중에는 **충돌 박스가 낮아짐** (음식 피하기 용이)

### 동작

- 슬라이딩 중: `girlY`는 그대로, 하지만 **충돌 박스 높이**를 줄임
- 슬라이딩 지속 시간: 약 0.5~0.8초 (30~48프레임 @ 60fps)
- 슬라이딩 종료 후: 다시 달리기 애니메이션으로 복귀

---

## 4. Pixel Lab 사용 팁

### 크로마키용 배경

- **라임 오렌지**: `#BFFF00`, `#ADFF2F`
- **녹색**: `#00FF00`
- 배경을 **단색**으로 유지하면 게임에서 크로마키로 투명 처리됩니다.

### 애니메이션 부드러움

- 프레임 간 변화를 **작게** 유지하면 부드럽게 보입니다.
- 슬라이딩은 빠른 동작이므로, 프레임 간격을 **짧게** (10~12fps) 설정하세요.

### 색상 일관성

- `run.mp4`와 비슷한 색상 팔레트를 사용하면 게임에서 자연스럽게 보입니다.
- 세라복: 네이비(`#1e3a5f`), 흰색(`#fff`)
- 피부: `#ffdbac`
- 머리카락: `#2c1810`

## 6. 게임 코드 추가 예시

슬라이딩 기능을 게임에 추가하려면:

1. **슬라이딩 비디오/이미지 로드** 추가
2. **슬라이딩 상태 변수** 추가 (`isSliding`, `slideFrames`)
3. **입력 처리**에 S키/아래 화살표 추가
4. **drawGirl()**에서 슬라이딩 애니메이션 표시
5. **충돌 박스** 조정 (슬라이딩 중 높이 감소)

자세한 코드는 `js/game.js`에 슬라이딩 기능을 추가하는 방식으로 제공됩니다.
