# 사용자 지시 #003: 배경 비율 수정 + 총알·음식 부서짐

**날짜**: 2025-01-25

---

## 지시 내용 (원문)

### 수정

> 1) 배경이미지가 **세로로 압착**되어 보인다. **배경이미지 원본비율을 그대로 살려서** 스크롤시켜.

### 추가 구현

> **마우스 왼쪽을 클릭**하면 **총알이 나가게** 만들고, **이 총알에 맞은 음식은 부서지게** 만들어.

---

## 정리

| 구분 | 항목 | 내용 |
|------|------|------|
| **수정** | 배경 | 원본 비율 유지, 세로 압착 방지. 비율 유지한 채 스크롤. |
| **추가** | 발사 | 마우스 왼쪽 클릭 → 총알 발사 |
| **추가** | 충돌 | 총알이 음식에 맞으면 → 해당 음식 제거 (부서짐) |
