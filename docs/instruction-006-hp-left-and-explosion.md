# 사용자 지시 #006: 체력 좌상단 + 탄환 히트 시 별 폭죽

**날짜**: 2025-01-25

---

## 지시 내용 (원문)

> - 🍟 3개로 체력 아이콘을 **화면 좌상단**으로 옮겨줘.
>
> - 탄환이 음식에 히트하면 **단순하지만 깔끔하게** **별모양 폭죽**이 터지는 효과를 넣어줘.

---

## 정리

| 구분 | 항목 | 내용 |
|------|------|------|
| **수정** | 체력 UI | 우상단 → **좌상단** |
| **추가** | 이펙트 | 탄환–음식 히트 시 **별모양 폭죽** (단순·깔끔) |
