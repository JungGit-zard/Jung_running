# 사용자 지시 #007: 탄환 히트 이펙트 — LED·디지털 스타일

**날짜**: 2025-01-25

---

## 지시 내용 (원문)

> 탄환이 히트되었을 때의 효과는 **반짝임이 가미된 LED 불빛** 같고, **진홍색**과 **녹색**과 **흩어지는 도트**의 **디지털적 이미지** 형상을 조합해서 표현해.

---

## 정리

| 요소 | 내용 |
|------|------|
| **LED 느낌** | 반짝임(sparkle)이 있는 밝은 핵(코어) |
| **색** | **진홍색**(crimson), **녹색**(green) |
| **형태** | **흩어지는 도트**(scattering dots) |
| **스타일** | **디지털/픽셀** 이미지 (도트, `fillRect` 등) |
