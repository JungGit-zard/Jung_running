# 사용자 지시 #002: 게임 기본 설계 (게임 시작 후 플레이 화면)

**날짜**: 2025-01-25

---

## 지시 내용 (원문)

> 게임시작 버튼을 누르고 들어가면
> - 주인공 캐릭터는 우선 **화면 왼쪽 끝 부근**에서 **달리기**를 하고 있어
> - 주인공은 **세라복을 입은 소녀**야
> - **주인공의 위치는 변하지 않으며**
> - 달리고 있는 소녀 **앞으로** 화면의 **오른쪽 끝에서 부터** **장애물인 각종 음식들**이 **다가온다**
> - 이 **음식물들을 점프로 피하는** 게임이야
> - **점프는 스페이스 바**를 눌러서 하는거야
> - 그리고 일단 **게임화면에 진입하면** 소녀는 달리고 있고 **배경그래픽이 스크롤**되며 흘러가고 있어
> - **배경이미지**는 **F:\cursor_project\graphic_resource\background.png** 를 사용해서 구현해
> - **시작!!!**

---

## 정리 (구현 체크리스트)

| 항목 | 내용 |
|------|------|
| 주인공 | 세라복( sailor uniform ) 입은 소녀 |
| 주인공 위치 | 화면 왼쪽 끝 부근, **고정** (이동 없음) |
| 주인공 동작 | 달리기 (런닝 애니메이션) |
| 장애물 | 각종 음식 |
| 장애물 등장 | 화면 오른쪽 끝에서 등장 → 소녀 쪽으로 다가옴 (좌측으로 이동) |
| 조작 | **스페이스 바** → 점프로 음식 회피 |
| 배경 | 게임 진입 시부터 **스크롤** (흘러가는 연출) |
| 배경 소스 | `graphic_resource/background.png` |

---

## 비고

- 게임 오버: 사용자 지시에는 미명시. 음식과 충돌 시 게임 오버 + 다시 하기로 구현.
- 점수: 사용자 지시에는 미명시. 생존 거리(스크롤량) 기반으로 선택 구현 가능.
